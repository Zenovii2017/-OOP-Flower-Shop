The program reads the year and, if desired, the number of countries
with the largest number of films this year

read_file - reads the file that you want to read, and the year.
Then rejects movies that are made not this year.
And reforming movies removes the extra parts.
return dictionary with keys of country and value with films.

change_adress - change adress in coordinates
input string return list of two int.

write_file - write dictionary in file 'result.txt'
return None.

create_map - create map of population, top country and population/area.
input dictionary of read_file and list of coordinate and int
return None

main - main part of program
input None
return None
